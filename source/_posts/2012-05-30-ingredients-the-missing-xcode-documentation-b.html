If you need to access the XCode documentation frequently, then Ingredients.app is for you. Give it a try: it is just easier to use than XCode’s builtin documentation browser.

But there is a gotcha: if you are on the latest XCode, both the official stable and beta binaries do not work! Ingredients being open sourced, someone fixed that bug already and pushed it at github, and the buys behind Ingredients.app merged it in already. So everyone is waiting for a new download.

Wait no longer! I compiled and built a binary for you. Until the Ingredients team comes with a new download package, you could just use this one. This is a debug build, with one exception: it skips question at startup, whether you would like Ingredients to forget all settings and cached documents.
